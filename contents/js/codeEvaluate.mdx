## 10?

정답을 맞추셨나요?
결과가 10인 이유는 연산자의 결합 순서와 평가 시점의 `i`값 때문입니다.

`i += x()`는 `i = i + x()`로도 나타낼 수 있습니다.

<mark>해당 라인이 평가되는 시점의 i는 0 입니다.</mark> 따라서 `i = 0 + x()`가 됩니다.

순서대로 적어보면
`i += x()` -> `i = i + x()` -> `i = 0 + x()` -> `i = 0 + 10`

이렇게 됩니다.

`x()`의 `return` 전에 콘솔을 찍어보면 이해하기 쉬운데요,

```javascript
function x() {
  i++;
  console.log(i);
  return 10;
}
i += x();
console.log(i);
// 1
// 10
```

`i++`가 실행되어 `i`가 `1`이 되었다가 `10`이 됩니다.

<u>이런 헷갈리는 코드를 작성하지 않는 게 중요하겠죠?</u>

### 번외

재미있는 건 연산의 순서를 변경하면 결과도 달라진다는 것.
`i = x() + i` 의 결과는 `11`입니다.

동일하게 순서대로 생각해보면

`i = x() + i` -> `i = 10 + i` (이때 i = 1) -> `i = 10 + 1`

이래서 11이 됩니다.
